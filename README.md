# Diagnóstico Comercial IA

Herramienta de diagnóstico comercial inteligente para profesionales de ventas e inmobiliaria.

## ¿Qué hace?

Permite a un usuario responder 15 preguntas sobre su negocio en menos de 5 minutos y obtener:

- 📊 **Diagnóstico ejecutivo** del estado comercial
- 🚨 **Cuellos de botella** detectados
- ⚡ **3 Prioridades estratégicas** personalizadas

## Tech Stack

- [Next.js 15](https://nextjs.org/) — Framework React
- [Tailwind CSS v4](https://tailwindcss.com/) — Estilos
- [Framer Motion](https://www.framer.com/motion/) — Animaciones
- [Lucide React](https://lucide.dev/) — Íconos

## Cómo correr localmente

```bash
cd comercial-ai-app
npm install
npx next dev -p 4000
```

Abrir [http://localhost:4000](http://localhost:4000) en el navegador.
